# Dark / Light Mode Toggle

This project implements a Dark and Light Mode toggle switch using HTML, CSS, and JavaScript.

## Features
- Dark and Light theme toggle switch
- Creative sun and moon icons
- Colors changed dynamically using JavaScript
- Smooth color transition effects
- Theme preference saved in localStorage
- Responsive and clean UI

## Technologies Used
- HTML
- CSS
- JavaScript
